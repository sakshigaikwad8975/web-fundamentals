
This folder contains basic JavaScript practice problems focused on core programming logic.

## Topics Covered
- If-else conditions
- Logical operators (AND, OR)
- Switch statements
- Number and string validation

## Problems Implemented
1. Even or Odd check  
2. Largest of three numbers  
3. Grade calculation system  
4. Username validation  

## Purpose
To build a strong foundation in JavaScript by solving basic logic problems and understanding control flow.

## Notes
All solutions are written in plain JavaScript without using advanced concepts, to strengthen fundamental thinking.
