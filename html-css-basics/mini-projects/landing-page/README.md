# Responsive Landing Page

This is a modern landing page built using HTML and CSS.

## Features
- Responsive layout using Flexbox
- Clean navigation bar with call-to-action button
- Hero section with heading and image
- Feature cards with icons
- Pricing section with highlighted plan
- Footer with useful links

## Technologies Used
- HTML5
- CSS3
- Font Awesome (for icons)

## Purpose
This project was built to practice creating a complete, real-world UI layout using only HTML and CSS.
